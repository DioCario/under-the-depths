# Under the Depths 

This is a simple game where you kill slimes
To run it, just clone it, cd into the folder, and run python main.py. You'll have to install python and pygame for it to run, and you may have to install a few other dependencies. If you don't know what that means, go to to https://www.python.org/downloads/, download and install the latest version for your operating system, then run:

```git clone https://github.com/DioCario/under-the-depths.git```

```cd under-the-depths```

```pip install pygame```

(if that command doesn't work, try ```py -m pip install pygame```, ```python -m pip install pygame```, or setup a virtual environment if you know how to)

```python main.py```
(you can also try ```python3 main.py``` if it fails)

WASD or Arrow keys to move
F or Space bar to attack
Shift to sprint

If you can't kill the slimes, try moving around a bit, the collision system is fairly buggy. I am unable to find a solution to it currently, so I'm leaving it as feature, to make the game harder.

If you look into the code, it's very messy. Most of the mess was caused from getting it to work in the web. I don't consider it a priority to go and optimize the code, especially since it runs perfectly fine on my 6 year old CPU.
